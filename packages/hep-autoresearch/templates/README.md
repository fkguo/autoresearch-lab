# templates/ (English)

This folder contains copy-paste templates for humans and for driving tool-using agents. They are not meant to be directly executable; they exist to standardize inputs and interactions.

Start here:
- `templates/idea_brief.md`: a structured “idea brief” filled by the user (for triage).
- `templates/prompt_idea_triage.md`: a prompt template to send to your tool-using agent (safe defaults + approval gates).

Chinese versions:
- `templates/idea_brief.zh.md`
- `templates/prompt_idea_triage.zh.md`

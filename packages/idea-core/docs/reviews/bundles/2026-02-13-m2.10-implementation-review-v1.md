# Review Bundle: 2026-02-13 M1.0+M1.6 + M2(min) Implementation v1

## 1) Goal / Scope / Non-Goals

### Goal
Implement the next executable stage of `idea-core` (standalone stdio JSON-RPC) with contract-first enforcement and auditable replay behavior.

### Scope (this bundle)
- Repo bootstrap as independent git repo.
- Contract source strategy (read-only vendored snapshot + explicit sync script).
- M1 tooling:
  - JSON parse / schema legality / OpenRPC minimal shape validation.
  - `$ref` closure check (OpenRPC + sibling schemas, fail-fast).
  - Drift guard (forbid OpenRPC schema copy/paste of complex schemas).
  - One-command execution (`make validate`, `make ci`).
- M2 minimal loop implementation (quality guard first):
  - `campaign.init`
  - `campaign.status`
  - `eval.run` (writes `scorecards_artifact_ref`)
  - `rank.compute` (failure-order + `effective_dimensions` + `scorecards_artifact_ref`)
  - `node.promote` (grounding + reduction gate)
  - idempotency replay + conflict handling (`idempotency_key_conflict` includes `payload_hash`)
- Required regression tests in this phase.

### Non-Goals (explicit)
- `search.step` and arXiv/INSPIRE ingest pipeline are not implemented in this phase.
- Strategy optimization / domain-depth heuristics are intentionally stubbed.

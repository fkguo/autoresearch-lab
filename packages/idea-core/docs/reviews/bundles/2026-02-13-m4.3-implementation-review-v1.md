## 3. Acceptance mapping (M4.3)

1. hepar command -> idea-core RPC translation only
   - Added command translation table (`hepar.idea.*` -> `campaign.init/status`, `search.step`, `eval.run`, `rank.compute`, `node.promote/get/list`).
   - `HeparSkillBridge.execute()` forwards params as-is to RPC transport without business-logic mutation.

2. Artifact persistence + replayability
   - Persists translated request artifact (`skill_bridge/requests/{command_id}.json`).
   - Persists RPC response artifact (`skill_bridge/responses/{command_id}.json`).
   - Maintains replay index (`skill_bridge/replay_index.json`) keyed by `command_id` + `request_hash`.
   - On replay hit with same hash: no second RPC call, return cached result + artifact refs.

3. Auditable control-plane events
   - Emits ledger events:
     - `skill_bridge.request_marshaled`
     - `skill_bridge.rpc_called`
     - `skill_bridge.response_persisted`
     - `skill_bridge.replay_hit`
   - Events include `command_id`, mapped `method`, request/response refs, and hash evidence.

4. Translation-layer boundary (no business logic backflow)
   - Bridge does only mapping, persistence, replay checks, and transport invocation.
   - Unknown command is rejected early with `ValueError` and no RPC side effects.

### `src/idea_core/hepar/skill_bridge.py`
```python
class HeparSkillBridge:
    def execute(self, command_request: dict[str, Any]) -> dict[str, Any]:
        rpc_method = self._translate_command(command_name)
        request_hash = payload_hash({"method": rpc_method, "params": params})

        replay_record = replay_index.get(command_id)
        if replay_record is not None:
            if replay_record["request_hash"] != request_hash:
                raise ValueError(f"command_id_conflict: {command_id}")
            response_path = self._artifact_ref_to_path(replay_record["response_ref"])
            response_payload = self._read_json(response_path, default={})
            self.control_plane_store.append_ledger_event("skill_bridge.replay_hit", ...)
            return {"replayed": True, "rpc_result": response_payload["rpc_result"], ...}

        self._write_json(request_path, marshaled_request)
        self.control_plane_store.append_ledger_event("skill_bridge.request_marshaled", ...)
        rpc_result = self.rpc_transport.call(rpc_method, params)
        self.control_plane_store.append_ledger_event("skill_bridge.rpc_called", ...)
        self._write_json(response_path, response_payload)
        self.control_plane_store.append_ledger_event("skill_bridge.response_persisted", ...)
```

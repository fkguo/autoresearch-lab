date_utc: 2026-02-18

1) Mathematical identity used (superconvergent / unsubtracted dispersion)

Assuming A(t) is analytic with a cut for t>4m_pi^2 and falls fast enough at infinity so that an unsubtracted dispersion is valid:

  A(t) = (1/pi) ∫_{4m_pi^2}^{∞} dt' ImA(t') / (t' - t).

Then:
  A(0)  = (1/pi) ∫ dt' ImA(t') / t'   (CSR)
  A'(0) = (1/pi) ∫ dt' ImA(t') / t'^2

In the campaign code we use dimensionless s := t'/m_pi^2, so the slope w.r.t s is:

  f1 := dA/ds|0 = (1/pi) ∫_{4}^{∞} ds ImA(s) / s^2 .

Therefore the implemented linear constraint is:
  ∫ ds ImA(s)/s^2 = pi * f1
with explicit subtraction of the fixed UV tail contribution beyond s0.

2) Low-energy slope input from arXiv:2507.05375 (NLO ChPT + TMD estimate)

From arXiv:2507.05375 Eq. (TMD):
  A^pi(t) = m_f2^2 / (m_f2^2 - t) = 1 + t/m_f2^2 + ...
so:
  dA/dt|0 = 1/m_f2^2 .

Convert to the dimensionless s=t/m_pi^2 used by the solver:
  f1 = dA/ds|0 = m_pi^2/m_f2^2 .

Numerical values used in configs (m_pi=0.13957 GeV, m_f2=1.275±0.020 GeV):
  f1 = 0.011982951111111114
  abs half-range tolerance (from m_f2 band) = 0.0003761207943301422

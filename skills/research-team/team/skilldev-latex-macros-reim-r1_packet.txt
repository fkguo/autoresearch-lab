Date: 2026-01-21
Scope: research-team skill — extend default LaTeX macro hygiene

Problem
- In real projects, Markdown math sometimes contains LaTeX macros \re and \im (Real/Imag part) copied from paper sources.
- These are typically custom macros and do not render in common Markdown math engines.
- Current defaults only forbid/expand \Rc/\Mc/\Cc/\cK.

Change
- Extend default forbidden macros + expansions to include:
  - re -> {\operatorname{Re}}
  - im -> {\operatorname{Im}}
- Update smoke test injection to include \re/\im so we regress this.

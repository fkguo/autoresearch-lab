- **NON-BLOCKING**: `_tex_escape` replaces `\` with `\\`. In LaTeX, `\\` inside normal text or `\texttt{}` generates a line break (or throws an error), not a literal backslash. If file paths or arbitrary text contain literal backslashes, this will break LaTeX compilation or render incorrectly. Consider using `\textbackslash{}` instead of `\\` for escaping `\`.
- **NON-BLOCKING**: `collect_run_result` iterates over `run_dir.rglob("*")` and eagerly computes the SHA256 hash for *every* file in the artifacts directory. If a run contains large datasets, model weights, or thousands of small intermediate files, this operation will become extremely slow. Consider adding a file size limit (e.g. skip files > 100MB) or an exclusion list for known heavy directories.

# autoresearch-meta

Autoresearch ecosystem governance, compatibility matrix, and cross-repo coordination.

## Scope

This repository is the policy/coordination SSOT across:
- `autoresearch-meta` (this repository)
- `hep-autoresearch`
- `hep-research-mcp`
- `idea-core`
- `idea-generator`
- `skills-market`

## Key Files

- `compatibility-matrix/ecosystem-manifest.json`: machine-readable compatibility SSOT
- `roadmap/ROADMAP.md`: staged rollout milestones
- `governance/GOVERNANCE.md`: ownership, release, and decision rules

python3 scripts/validate_manifest.py
